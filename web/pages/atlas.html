Sacred Topology Atlas

shrine_master.tsv の 全神社を対象に、社格でフィルタしながら神社分布を地理可視化。 初期表示は 名神大社。座標を持つ神社のみ地図に描画され、座標未登録件数も明示します (DISC-008)。

🗺️ 本地図は地理的相関を示しますが、因果関係を意味しません

Narrative Intoxication 警告 (DISC-008 採択):

  • 「線が引ける」「近くにある」「分布が似ている」だけで歴史的因果を断定しないでください
  • 地図は探索の手がかりであり、解釈の検証は graph query + 一次史料への遡及で行ってください
  • 座標カバレッジの偏り注意: 現在 shrine_master の座標登録は出雲・伊勢・主要一宮等に偏在し、全神社の分布を代表しません。未登録社が地図に無いことは「存在しない」を意味しません
  • coordinates_accuracy / visualization_confidence を popup に表示。座標精度の幻想を避けてください

詳細: DISC-008 sacred topology 解決報告

visualization confidence (DISC-008 採択、Phase 2 実装済)

各 marker の visualization_confidence field で根拠強度を明示しています (popup に表示):

  • archaeological: 考古学根拠 (出土品・遺構との対応) — Phase 3 で適用
  • textual: 文献根拠 (記紀・延喜式・社伝) — 古代社格保持社 25 件に適用
  • inferred: 妥当な推論 (古代海運ルート推定等) — 著名社 3 件に適用
  • speculative: 検証困難仮説 (中央構造線神社配置論等) — Phase 3 で適用

Narrative intoxication 警告再掲: visualization_confidence が textual でも、 個別の地理的相関は因果関係を意味しません。
