# 日本神話・神道史データベース

氷河期・旧石器・縄文時代から鎌倉時代までの日本列島に関する **神話・神社・豪族・皇統・祭祀・地方伝承・考古学** を統合する巨大知識体系を、Excel 互換の TSV として整理する編纂プロジェクト。

## 目次(編)

- 出雲編 (`docs/出雲編/`)
- 伊勢編(予定)
- 諏訪編(予定)
- 鹿島・香取編(予定)
- 宗像編、熊野編、石上編、三輪編、宇佐編、春日編(予定)

## 出力形式

- **TSV (タブ区切り)** — Excel に直接貼り付け可能
- 全 29 列の固定スキーマ(編纂規程 [`CLAUDE.md`](./CLAUDE.md) §3.2 参照)
- ID は `<編コード>-<3桁連番>` (例: `IZM-001`)
