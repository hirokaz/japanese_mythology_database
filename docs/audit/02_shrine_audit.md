## 3. 共通指摘(横断)

### 3.1 主祭神 DEI 未マスター化による空欄

| SHR | 神社 | 必要 DEI |
|---|---|---|
| SHR-003 | 伊勢外宮 | DEI-031 豊受大神 |
| SHR-011 | 石上神宮 | DEI-042 布都御魂大神 |

| SHR-016 | 宇佐神宮 | DEI-035 八幡神 + 比売大神 + 神功皇后 |

→ 監査 #2(deity audit)で指摘した未収録神の優先追加と連動。

### 3.2 創建年代の二段表記推奨

ほぼ全ての主要神社で、社伝年代と学術推定年代の **両方を併記** することを推奨:
- 形式案: `founding_year_estimated` を `社伝: XXX 年 / 学術: XXX 期` 形式に
- または別カラム `founding_year_legend` / `founding_year_academic` を新設

### 3.3 神仏習合期の名称履歴

明治神仏分離で改称した神社の旧称履歴が `old_names` で網羅されているか:
- SHR-017 八坂(○ 祇園社・祇園感神院)
- SHR-016 宇佐(△ 宇佐八幡宮 + 宇佐弥勒寺[神宮寺]の併存履歴未記録)
- SHR-005 熊野本宮(△ 熊野権現の名称変遷未記録)

→ 神宮寺・別当寺の関係を含む神仏習合史の補完が将来課題。

## 4. 監査総括

### 4.1 重要度別の指摘件数

| 重要度 | 件数 | 主要対応 |
|---|---|---|
| 高 | 11 | 主祭神 DEI 紐付け、宗像沖ノ島 SITE 立項、宇佐比売大神三説併記、賀茂氏二系統分離 |
| 中 | 21 | 創建二段表記、SCH_MIX 修正、四宮三宮の正規化 |
| 低 | 5 | 軽微な表記補完 |

### 4.2 推奨対応の優先順位

1. **deity_master 拡張**(監査 #2 連動): 豊受・八幡・布都御魂・賀茂建角身・賀茂玉依等の追加
2. **創建年代の二段表記**: 全 28 神社で社伝/学術の併記
3. **神仏習合履歴**: 神宮寺・別当寺の旧称を `old_names` に
4. **三宮・四宮の正規化**: 諏訪大社・宗像大社・伊勢神宮の子社レコード化検討

### 4.3 推奨出典(共通)

- **神社本庁**: 全国神社の正式社格・祭神情報の照合
- **NDLデジタルコレクション**: 続日本紀、延喜式、各風土記、中世神道書
- **国立歴史民俗博物館**: 古代祭祀の考古学的研究
- **島根県立古代出雲歴史博物館**: 出雲関連神社の最新研究
- **各神社公式**: 一次的社伝記録(`official` レベル)
