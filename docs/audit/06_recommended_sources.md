## 9. アンチ推奨出典(注意)

以下は **本DBの一次出典として用いない** ことを推奨:

- 個人ブログ・SNS(speculative レベル、特例除く)
- 内容の出典がない出版物(タブロイド系)
- 学術検証なき民間研究書(`speculative` カテゴリ)
- AI 生成コンテンツの直接引用(検証なし)
- 寺社の観光案内パンフレット(社伝としては OK だが、official レベル止まり)

→ 上記は内容として参照しても良いが、最終的な出典としては避け、より上位の出典に遡ることを推奨。

## 10. 推奨出典の今後の拡張

1. **第一段階**: 公的機関・主要神社・国立博物館の URL リンクを各 master の `source_reference` に追加
2. **第二段階**: 学術論文の DOI / J-STAGE リンクをマスター注釈に
3. **第三段階**: 国会図書館デジタル史料へのパーマリンク化(史料の特定箇所)

これらは後続フェーズで段階的に整備。
