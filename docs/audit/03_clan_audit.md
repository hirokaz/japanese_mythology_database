## 4. 共通指摘(横断)

### 4.1 ancestor_deity_id の DEI 未マスター化

現行 clan_master で ancestor_deity_id が「-」になっている氏族:
- CLN-010 蘇我氏
- CLN-011 大伴氏 → アメノオシヒノミコト(未マスター)
- CLN-012 紀氏 → 天道根命(未マスター)

→ 監査 #2(deity audit)の DEI 拡張と連動。

→ 阿蘇・隼人・蝦夷・出羽・吉備・海部・尾張・紀伊国造等の地方豪族の追加で 50% 以上を目標。
