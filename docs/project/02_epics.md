# Epic 一覧 — 32 Epics

巨大テーマ単位で Epic を定義する。各 Epic は **複数 Milestone** を含み、**複数 Issue** が紐づく。Epic は GitHub の Project Board または `epic:*` ラベルで管理する。

## 0. Epic 番号体系

```
EPIC-XX  : 全 Epic 通し番号
編コード : CLAUDE.md §6 と同期(IZM, ISE, SUW, ...)
タイプ   : R(地域)/ N(ネットワーク)/ M(マスター)/ X(横断)/ G(グラフ)/ A(監査)
```

## 2. 各 Epic 詳細

### EPIC-01. 出雲祭祀ネットワーク
- **目的**: 出雲大社・熊野大社を中心とする祭祀権力史の網羅
- **主要ノード**: 大国主、スサノオ、アメノホヒ、オオナムチ、コトシロヌシ、タケミナカタ、出雲国造、千家、北島、意宇、神門、出雲、安来、加茂岩倉、荒神谷、西谷
- **主要 relation**: enshrined_at(大国主→出雲大社)、ancestor_deity_of(アメノホヒ→出雲国造)、participated_in(国譲り)、archaeologically_linked(銅鐸祭祀⇄国譲り)
- **状態**: ✅ 完了(`docs/出雲編/`)

### EPIC-02. 伊勢祭祀ネットワーク
- **目的**: 内宮(アマテラス)・外宮(豊受)・所属社家(荒木田・度会)・斎宮制度・伊勢神道
- **主要ノード**: アマテラス、豊受大神、荒木田氏、度会氏、斎宮、宇治山田、神宮式年遷宮、皇大神宮儀式帳
- **主要 relation**: primary_deity_of(アマテラス→内宮)、performed_at(神嘗祭)、authored_by(神宮儀式帳→中臣)、descended_from(荒木田→天児屋根)
- **状態**: 未着手

- **状態**: 未着手
