# 地方伝承研究ディレクトリ

本ディレクトリは「日本神話・神社・豪族・皇統・祭祀・考古学統合データベース」の **地方伝承・地方神話・地域祭祀ネットワーク** を担当する。

## 担当領域

- 地方ごとの神話構造
- 地方祭祀圏
- 神社ネットワーク
- 豪族ネットワーク
- 海人族ネットワーク
- 製鉄民ネットワーク
- 山岳信仰圏
- 神仏習合ネットワーク

## 設計の出発点

監査 #9(地方伝承軽視)で指摘された **構造的偏り**(東北0/沖縄0/九州2/四国0、海人族・修験道欠落)への対応として独立ディレクトリを設立。

「中央史観だけで整理しない / 地方伝承を軽視しない」を最優先原則とする。

## ID 体系の継続

新規エンティティは既存 ID 体系を継続使用:

- 神: `DEI-NNN`(deity_master.tsv 拡張)
- 神社: `SHR-NNN`(shrine_master.tsv 拡張)
- 豪族: `CLN-NNN`(clan_master.tsv 拡張)
- 神話: `MYTH-NNN`(本ディレクトリで新規 master 化)
- 関係: `RLN-NNNNNN`(relations 拡張)
- 仮説: `HYP-NNN`(hypothesis_master 拡張)
- 地方テーマ: `THM-NNN`(本ディレクトリで新設)
