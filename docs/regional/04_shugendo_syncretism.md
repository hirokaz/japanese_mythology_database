# 修験道・神仏習合DB

## 1. 修験道の概要

### 1.1 起源と展開

- **開祖伝**: 役小角(役行者、7世紀末) — 大峰山で修行
- **発展**: 平安期に天台宗・真言宗との習合により体系化
- **中世**: 全国の山岳霊場で修験道が確立
- **近世**: 当山派(真言系)・本山派(天台系)の二大派閥
- **明治**: 神仏分離令(1868)+ 修験道廃止令(1872)で大打撃 → 多くの霊場が神社化

### 1.2 修験道の特徴

- **山岳信仰**: 山そのものを神仏の住処とする
- **神仏習合**: 神=仏の権現とする本地垂迹説
- **山伏**: 山中での修行者(山伏装束、ホラ貝)
- **三世観**: 過去・現在・未来の三世を山に重ねる

## 2. 主要修験霊場

### 2.1 出羽三山(月山・羽黒・湯殿山)

- **地域**: 山形県(陸奥/出羽国)
- **三世構造**: 羽黒=現世、月山=死後、湯殿山=未来
- **開祖**: 蜂子皇子(崇峻天皇皇子、伝)
- **神仏習合**: 観音(羽黒)・阿弥陀(月山)・大日(湯殿山)
- **特徴**: 山伏修行の最高峰、即身仏文化(湯殿山系)
- **明治の影響**: 神仏分離で月山・羽黒・湯殿山神社として神社化

### 2.2 白山(石川・富山・岐阜)

- **地域**: 加賀・越前・美濃国境(白山連峰)
- **主神**: ククリヒメ(白山比咩大神)
- **本地仏**: 十一面観音
- **開祖**: 泰澄(伝、奈良期)
- **特徴**: 北陸の総鎮守、白山信仰の全国拡散(白山神社2700+)
- **奥宮**: 白山比咩神社奥宮(山頂)

### 2.3 立山(富山)

- **地域**: 越中国(立山連峰)
- **主神**: 立山権現(伊邪那岐神=本地阿弥陀如来)
- **特徴**: 立山地獄(地獄谷)信仰、女人禁制の歴史
- **開祖**: 佐伯有頼(伝、大宝元年)
- **明治の影響**: 雄山神社・立山若宮として神社化

### 2.4 戸隠(信濃)

- **地域**: 信濃国(戸隠山)
- **主神**: 天手力雄命(戸隠岩戸開きの神)
- **本地仏**: 阿弥陀(奥社)・聖観音(中社)・地蔵(宝光社)
- **特徴**: 戸隠九頭龍信仰(山中の岩窟祭祀)
- **江戸期**: 衆徒3000、修験の一大拠点

### 2.5 富士山(駿河・甲斐)

- **地域**: 駿河国・甲斐国(富士山)
- **主神**: 木花咲耶姫(浅間大神)、コノハナサクヤビメ
- **本地仏**: 大日如来(浅間大菩薩)
- **特徴**: 山頂祭祀、富士講の全国展開(近世)
- **総本宮**: 富士山本宮浅間大社(静岡県富士宮市)

### 2.6 大峰山・吉野(大和)

- **地域**: 大和国(大峰山系)
- **主神**: 蔵王権現(役小角感得、本地仏=釈迦・観音・弥勒)
- **特徴**: 修験道発祥地、女人禁制の伝統
- **金峯山寺**: 蔵王権現の総本山
- **吉野**: 桜の名所、金峯神社、吉水神社

### 2.7 熊野三山(紀伊)

- **地域**: 紀伊国
- **三山**: 熊野本宮大社(本地阿弥陀)、熊野速玉大社(本地薬師)、熊野那智大社(本地観音)
- **特徴**: 熊野権現信仰、補陀落渡海(浄土信仰)
- **熊野修験**: 山伏文化の中心、全国展開
- **平安期**: 院政期に天皇・上皇の熊野御幸が頻繁化

### 2.8 英彦山(豊前)

- **地域**: 豊前国・筑前国(英彦山)
- **主神**: 天忍骨命(天忍穂耳命=正勝吾勝勝速日天忍穂耳尊)
- **本地仏**: 阿弥陀如来
- **特徴**: 九州修験の中心、求菩提山と並ぶ
- **江戸期**: 衆徒800、日本三大修験

### 2.9 求菩提山(豊前)

- **地域**: 豊前国(求菩提山)
- **特徴**: 英彦山と並ぶ豊前修験
- **明治の影響**: 修験道廃止で衰退

### 2.10 木曽御嶽

- **地域**: 信濃国・美濃国境(御嶽山)
- **主神**: 国常立尊・大己貴命・少彦名命(御嶽大神)
- **特徴**: 近世講社「御嶽講」の全国展開
- **御嶽教**: 明治に独立宗教化

## 3. 神仏習合構造

### 3.1 本地垂迹説

「神」は「仏」が衆生救済のために仮の姿で現れたとする思想。

| 神 | 本地仏 | 主な神社 |
|---|---|---|
| アマテラス | 大日如来 | 伊勢神宮(両部神道で) |
| ハチマン | 阿弥陀如来 | 宇佐神宮、石清水八幡宮 |
| クマノ本宮(家都美御子) | 阿弥陀如来 | 熊野本宮大社 |
| クマノ新宮(速玉) | 薬師如来 | 熊野速玉大社 |
| クマノ那智(夫須美) | 千手観音 | 熊野那智大社 |
| サンノウ(大山咋) | 釈迦如来 | 日吉大社 |
| 蔵王権現 | 釈迦・観音・弥勒 | 金峯山寺 |
| 浅間(コノハナサクヤ) | 大日如来 | 浅間大社 |
| ハクサン(ククリヒメ) | 十一面観音 | 白山比咩神社 |
| カスガ(タケミカヅチ) | 不空羂索観音 | 春日大社 |
| イチキシマヒメ | 弁才天 | 厳島神社・竹生島 |
| スサノオ | 牛頭天王(本地は薬師如来) | 八坂神社・氷川神社 |

### 3.2 神宮寺・別当寺

各主要神社に併設された仏教寺院。神仏習合期の主要な祭祀拠点。

| 神社 | 神宮寺・別当寺 |
|---|---|
| 伊勢神宮 | 大神宮寺(神宮寺、平安期に廃止) |
| 宇佐神宮 | 弥勒寺 |
| 石清水八幡宮 | 護国寺 |
| 熊野三山 | 熊野山新宮神蔵寺等 |
| 春日大社 | 興福寺 |
| 出雲大社 | 鰐淵寺(出雲西部) |
| 諏訪大社 | 神宮寺 |
| 日吉大社 | 延暦寺(山王神道) |
| 大神神社 | 大御輪寺・平等寺 |

### 3.3 明治神仏分離の影響

- **1868年(慶応4年/明治元年)**: 神仏分離令
- **1872年(明治5年)**: 修験道廃止令
- **影響**:
  - 各地の権現が神社化(熊野権現→熊野速玉/那智/本宮大社など)
  - 神宮寺・別当寺の廃寺(伊勢の大神宮寺は早期廃止だが多くは明治期)
  - 修験者(山伏)の還俗または僧化
  - 仏像・仏具の破却(廃仏毀釈)
  - 一部の修験霊場の神社移行(出羽三山等)

## 4. 修験ネットワーク

```
本州中北部
- 出羽三山(山形)
- 飯豊山・鳥海山(東北)
- 戸隠・八海山(信濃・越後)
- 三峰山(秩父)
- 白山(加賀・越前・美濃)
- 立山(越中)

中部
- 富士山(駿河・甲斐)
- 御嶽山(信濃・美濃)

西日本
- 大峰山(大和) - 修験総本山
- 熊野三山(紀伊) - 神仏習合の中心
- 比叡山(山城・近江、天台山岳)
- 高野山(紀伊、真言山岳)

九州
- 英彦山(豊前)
- 求菩提山(豊前)
- 国東(豊後、神宮寺集中地)
```

→ 山岳の修験ネットワーク+神宮寺の都市信仰圏が、中世日本の宗教地理を形成。
