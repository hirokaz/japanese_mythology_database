# 海人族・製鉄民ネットワーク

## 1. 海人族ネットワーク

### 1.1 阿曇氏(あづみ)

- **本拠**: 北部九州志賀島(志賀海神社)
- **祖神**: 綿津見三神(底津/中津/上津)、安曇磯良
- **海洋ネットワーク**: 玄界灘・対馬・壱岐・朝鮮半島、瀬戸内海、日本海
- **内陸移住**: 信濃国安曇野(現長野県安曇野市)— 海人族の内陸進出の代表例
- **皇統との関係**: 神功皇后の三韓征討随伴(伝承)
- **系譜**: 新撰姓氏録「海神綿積豊玉彦命の子穂高見命の後」(L1)

### 1.2 住吉氏(津守氏)

- **本拠**: 摂津住吉(住吉大社)
- **祖神**: 住吉三神(底筒男/中筒男/表筒男)
- **役割**: 古代の難波津・住吉津からの海上交通祭祀
- **皇統との関係**: 神功皇后伝承の中軸氏族
- **特徴**: 海路守護神(航海の事故防止、新羅・任那航路守護)

### 1.3 宗像氏(宗形氏)

- **本拠**: 筑前宗像(宗像大社)
- **祖神**: 宗像三女神(田心姫/湍津姫/市杵島姫)
- **海洋ネットワーク**: 沖ノ島→大島→田島の三宮制で玄界灘北上ルート祭祀
- **国家祭祀**: 沖ノ島祭祀遺跡(古墳期〜平安期)で大和政権の対外祭祀を主導
- **対外交渉**: 朝鮮半島航路(新羅・任那・百済)の祭祀

### 1.4 海部氏(あまべ)

- **本拠**: 丹後(籠神社)
- **祖神**: 彦火明命(ホアカリ)
- **特徴**: 元伊勢の伝承を保持、海部氏系図(国宝)で系譜が古代まで遡る
- **海洋ネットワーク**: 日本海ルート(若狭・丹後・但馬)

### 1.5 尾張氏(おわり)

- **本拠**: 尾張(熱田神宮、真清田神社)
- **祖神**: 天火明命(海部氏と同祖伝)
- **特徴**: 草薙剣の継承者、東海道の海陸交通の要
- **皇統との関係**: ヤマトタケル妻ミヤズヒメの父

## 2. 製鉄・金属民ネットワーク

### 2.1 出雲製鉄

- **地域**: 奥出雲(斐伊川流域、船通山・鳥髪)
- **時期**: 中世以降のたたら製鉄が確実史料、古代の在地製鉄証拠は乏しい
- **神話**: ヤマタノオロチ製鉄民征服譚説(L4)、草薙剣の出雲産鉄剣説(L3)
- **氏族**: 山内氏(鉄師)、絲原氏、桜井氏(近世)
- **遺跡**: 菅谷たたら(国指定重要有形民俗文化財)
- **製鉄民の祭祀**: 金屋子神(東出雲・島根)、金山彦神

### 2.2 吉備製鉄

- **地域**: 備前・備中・備後(岡山県・広島県東部)
- **特徴**: 古代の確実な製鉄遺跡(古墳期〜飛鳥期)
- **氏族**: 吉備氏(吉備上下道氏、吉備津彦命系)
- **神社**: 吉備津神社、吉備津彦神社(備前一宮、備中一宮)
- **神格**: 吉備津彦命(温羅退治)、温羅伝承(渡来製鉄民か)

### 2.3 九州製鉄

- **地域**: 北部九州(古代の砂鉄産地、朝鮮半島と近接)
- **特徴**: 朝鮮半島の伽耶製鉄技術の流入経路
- **海人族ネットワーク**: 阿曇・宗像が鉄の流通を支える可能性

### 2.4 渡来系技術

- **秦氏**: 治水・織物・酒造(製鉄も部分的)
- **漢氏(東漢氏)**: 軍事・武器製造
- **西文氏**: 文書・記録
- **百済王氏**: 製鉄・金工

### 2.5 鍛冶神・製鉄神

- **金山彦神**: 金属神の中核(美濃国南宮大社)
- **金山姫神**: 金山彦の対偶神
- **金屋子神**: 出雲・伯耆の製鉄神(金屋子神社、奥出雲)
- **天目一箇神**: 鍛冶神(『古語拾遺』、忌部氏祖伝)
- **稲荷神**: 鍛冶守護神化(伏見稲荷との関連)

### 2.6 鉄と武神信仰

- **草薙剣**: ヤマタノオロチから出る、皇統の三種神器
- **十握剣・天羽々斬**: 神話の剣
- **七支刀**: 石上神宮(369年銘)
- **物部氏**: フツヌシ祖神、剣の祭祀
- **中臣氏**: タケミカヅチ(剣神)祖神

## 3. 海人族 × 製鉄民の交差点

- 阿曇族の信濃移住(安曇野)は、北部九州の鉄文化が信濃まで伸びた可能性
- 宗像族の沖ノ島祭祀遺物には鉄器多数(朝鮮半島系)
- 出雲の銅鐸文化(IZM-006〜008)が朝鮮半島起源(部分的)とすると、海人族の流通ルートと連動
