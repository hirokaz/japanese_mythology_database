# 地方伝承研究の方法論

## 1. 中央史観回避戦略

### 1.1 「中央 vs 地方」ではなく「複数視点の総合」

`docs/audit/08_regional_audit.md` で指摘されたように、本DB の最大弱点は **収録範囲の地理的・文化的偏り** だった。これは「出雲固め後に他地域へ」という編纂順序が **中央集権的な思考** に陥っていたためでもある。

本研究では原則を以下に転換する:

- **中央史(古事記書紀)を否定するのではない**
- **地方視点を独立した「もう一つの正史」として扱う**
- 同じ事象に対して **複数の視点を並列保持**

例: 国譲り神話
- **中央視点**(古事記書紀): 大国主が国を譲った
- **出雲視点**(出雲国造神賀詞): アメノホヒが大国主を慰撫し、御子神を皇都守護に配置
- **諏訪視点**(諏訪大社社伝): タケミナカタは敗れたのではなく退避し、諏訪を「出ない」と誓ったが、それは敗北ではなく在地神格としての確立
- **地方視点**(出雲西部神門郡・宗像沖ノ島): 中央化される以前の在地祭祀の物質的痕跡

→ 全て独立に保持し、`relations` で結合する。

### 1.2 社伝・地方伝承の独立価値

社伝(神社縁起)は学術的には **二次的な史料** だが、地方研究では:

- 地域共同体が **何を記憶しているか** の一次史料
- 中央史料に欠落する出来事を補完する可能性
- 神社の自己認識(=祭祀権の正統性主張)の鏡

→ 学術通説と社伝を **混同せず、両方を保持**(source_reliability で区別)。

### 1.3 海人族・修験道の重視

監査 #9 で指摘された **海人族・修験道領域の欠落** を本研究で集中補完する。これは:

- 海人族 = 古代国際交易の主体(沖ノ島祭祀、住吉海路、阿曇海洋ネットワーク)
- 修験道 = 中世神仏習合の中核(神道DBから漏らすと中世以降の宗教史が描けない)

両者なくして「日本神話・神道史」のDB は完成しない。

## 2. 「祭祀圏」という分析視点

### 2.1 祭祀圏の定義

本研究では「祭祀圏(さいしけん)」を次のように定義:

> 一定の **地理的範囲** で、共通の **神格・祭祀・神社ネットワーク・氏族支配** を持つ宗教圏。

例:
- **出雲祭祀圏**: 出雲国造を中心に、杵築・熊野・佐太・須佐・美保等の神社ネットワーク、大国主・スサノオ祭祀、神在月の神迎神事
- **諏訪祭祀圏**: 諏訪四宮+全国諏訪神党、御柱祭、タケミナカタ・八坂刀売・ミシャグジの三層信仰
- **宗像祭祀圏**: 沖ノ島祭祀、宗像氏の海上交易、住吉・阿曇との海人族ネットワーク
- **熊野祭祀圏**: 熊野三山+全国熊野神社、修験道、補陀落信仰、本地垂迹
- **三輪祭祀圏**: 大神神社+三輪山、大物主祭祀、葛城賀茂氏

### 2.2 祭祀圏分析の三軸

| 軸 | 内容 | 関連エンティティ |
|---|---|---|
| 神格軸 | どの神を祀るか | deity / title |
| 制度軸 | どの神社・氏族が祭祀権を持つか | shrine / clan / ritual |
| 地理軸 | どこで祭祀が行われるか | region / site |

→ 知識グラフでは三軸の交点を `enshrined_at` / `performed_at` / `located_in` 関係で表現。

### 2.3 「神を祀る権利 = 地域支配」の視点

古代日本では祭祀権が地域支配と結びつく。出雲国造の出雲祭祀権、宗像氏の沖ノ島祭祀権、阿蘇氏の阿蘇山祭祀権など。

→ `clan` ↔ `shrine`(祭祀権)の関係を `controlled_by` / `served` 等で記録する。

## 4. 出力データの最低要件

### 4.1 myth(地方神話DB)
- 中央視点と地方視点を **同一 myth_id 内で異伝として保持**(`variants` カラム)
- motif_category を必須化(蛇神/海神/山神/製鉄/狩猟/王権 等)

### 4.2 shrine(社伝DB)
- founding_tradition(社伝)と shrine_history(学術)を **別カラム** で保持
- syncretism_history(神仏習合史)を独立カラム化

### 4.3 themes(地方伝承テーマ集)
- 「中央史で軽視」かつ「地方で重要」の二条件を満たすテーマのみ採択

### 4.4 relations
- region 列を必須化(地理スコープを明示)
- period 列で時代を限定

## 5. 重要原則の再確認

- 中央史観だけで整理しない
- 地方伝承を軽視しない
- 神話を単純否定しない
- 学術説と伝承を分離(source_reliability 必須)
- 社伝を重要視
- 神仏習合を重要視
- 「祭祀圏」という視点を重視
- 「神を祀る権利が地域支配と結びつく」視点を重視
- 山岳・海洋・製鉄信仰を重視
- relation 構造を重視
- 将来 Neo4j 化を前提
