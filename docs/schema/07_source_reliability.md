# 出典強度設計

各レコードの出典(source_reference)に対し、信頼性を 6 段階で分類する。

## 1. レベル一覧

| Level | コード | 名称 | 定義 |
|---|---|---|---|
| 1 | `primary` | 一次史料 | 当該事象に最も近い時点で書かれた現存史料 |
| 2 | `official` | 公文書・公記録 | 朝廷・社家の公的文書、社伝の正式記録 |
| 3 | `academic` | 学術研究 | 査読論文・学会誌・専門書 |
| 4 | `local_tradition` | 地方伝承 | 地域に伝わる口碑・郷土史記録 |
| 5 | `folklore` | 民俗学的記録 | 民俗調査による聞き取り、習俗記録 |
| 6 | `speculative` | 仮説・私説 | 個人研究・大胆仮説・未検証説 |

## 2. 各レベルの詳細

### 2.1 primary(一次史料)

- **定義**: その事象が書かれた最古に近い文献。
- **用途**: 国譲り神話 → 古事記・日本書紀(8世紀)。神賀詞 → 続日本紀・延喜式。
- **使用ルール**:
  - 文献名 + 巻・段を可能な限り記録
  - 写本系統が複数ある場合は系統名を備考に
- **例**:
  - 古事記(712)、日本書紀(720)、出雲国風土記(733)、続日本紀(797)、延喜式(927)、万葉集

### 2.2 official(公文書・公記録)

- **定義**: 国・地方公権力・社家による公式記録。primary に至らないが公的根拠あり。
- **用途**: 平安〜中世の社家文書、寛文造営記、近世の藩記録。
- **使用ルール**:
  - 発行主体を必須記録
  - 私文書と区別
- **例**:
  - 千家家文書、北島家文書、寛文造営記、松江藩公記録

### 2.3 academic(学術研究)

- **定義**: 学術機関・査読誌・専門書。
- **用途**: 近代以降の研究論文、考古学発掘報告、神話学・民俗学・歴史学の専門書。
- **使用ルール**:
  - 著者名と発行年を記録
  - 学会の主流説/少数説の区別を notes に
- **例**:
  - 津田左右吉、岡田精司、佐原真、瀧音能之 等の学術論文
  - 島根県教育委員会発掘報告

### 2.4 local_tradition(地方伝承)

- **定義**: 地域の口碑・郷土誌・地名由来譚。
- **用途**: 風土記の段階を過ぎた、近世以降の地方記録。
- **使用ルール**:
  - 採録時期と採録者を記録
  - 学術検証の有無を区別
- **例**:
  - 江戸期の郷土誌、地方寺社縁起、地名由来譚

### 2.5 folklore(民俗学的記録)

- **定義**: 民俗学的手法で採録された習俗・口碑。
- **用途**: 柳田國男・折口信夫以降の民俗学資料。
- **使用ルール**:
  - 採録者・採録地・採録年を記録
- **例**:
  - 民俗調査報告、神事の現代記録、家伝・口承

### 2.6 speculative(仮説・私説)

- **定義**: 個人研究・大胆仮説・未検証説・SNS等の現代的言説。
- **用途**: 海面上昇記憶仮説、製鉄渡来民征服説 等の大胆仮説。
- **使用ルール**:
  - 提唱者を必ず記録
  - hypothesis_master.tsv にも立項
  - confidence_level は通常 D-E
- **例**:
  - 個人研究書、ブログ記事、ファン理論

## 3. 既存「史料種別」「史実性レベル A-E」との対応

### 3.1 史料種別との対応(参考)

既存編 TSV の「史料種別」(正史/神話/風土記/社伝/神社縁起/地方伝承/系譜資料/考古学/民俗学/中世神道/近世資料/近代研究/現代研究/仮説)は、本書のレベルと交差する **メタデータ** として残す。

| 既存史料種別 | 主な対応レベル |
|---|---|
| 正史(古事記、日本書紀、続日本紀) | primary |
| 神話、風土記 | primary |
| 社伝、神社縁起 | official(公式)/ local_tradition(伝承部) |
| 地方伝承 | local_tradition |
| 系譜資料(先代旧事本紀 等) | primary(平安初期成立部) or speculative(後世補加部) |
| 考古学(発掘報告) | academic |
| 民俗学 | folklore |
| 中世神道(神道集 等) | official(中世期) or speculative(中世神話の創作) |
| 近世資料 | local_tradition or academic |
| 近代研究、現代研究 | academic |
| 仮説 | speculative |

### 3.2 史実性レベル A-E との対応

| 史実性レベル | 主な source_reliability_level | 補足 |
|---|---|---|
| A(文献・考古双方で裏付け) | primary + academic | 複数強度からの裏付け |
| B(文献記録あり、一定の蓋然性) | primary or official | |
| C(神話・伝承として重要、史実性未確定) | primary(神話) or local_tradition | |
| D(後世の創作・政治的編集) | speculative or official(中世神道) | |
| E(仮説・推測) | speculative | hypothesis_master 連結 |

→ **史実性レベル(A-E)** は事象の確からしさ、**source_reliability** は出典の強度。両者は独立して付与する。

## 4. 運用ルール

### 4.1 複数出典の優先順位

同じ事象に対し複数出典がある場合、最強の出典強度をレコードの代表 source_reliability に。
他の出典は relations.tsv の `mentioned_in` で個別記録。

### 4.2 出典の引用形式

- **必須**: 出典名 + (成立年)
- **推奨**: 巻・段・条の特定(例: 「日本書紀 神代下 第九段 一書第二」)
- **学術論文**: 著者(出版年)「論文名」誌名 巻号

### 4.3 「不明」の扱い

出典不明の主張は:
- source_reliability = `speculative`
- confidence_level = `E`
- notes に「出典不明、検証要」を明記

### 4.4 レコード単位 vs 関係性単位

- **マスター TSV**(node 単位): notes 列で出典を補足
- **relations.tsv**(edge 単位): `source_reference` 列を必須化
- **編 TSV**(年表): 既存の「出典」「史料種別」カラムを継続使用

---

## 5. 6 レベルと既存仮説レベル(0-5)の関係

仮説レベル 0-5(`docs/templates/columns.md` §仮説レベル)は事象の **解釈の強さ** を示し、source_reliability は **出典の強さ** を示す。両者は独立。

例: 「銅鐸祭祀終焉=国譲り神話対応説」
- source_reliability = `academic`(津田左右吉 等の学術論文に基づく)
- 仮説レベル = `2`(複数研究者が言及する推定)
- confidence_level = `B`(文献+考古の両面)

→ 出典自体は強い(学術論文)が、その内容は仮説レベル 2。両軸で記録。
