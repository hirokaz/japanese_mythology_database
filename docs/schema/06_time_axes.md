# 時間軸設計(三層モデル)

神話・神道史データベースにおける **時間情報の最大の罠** は、以下の三つの時間が無自覚に混在することである。

1. **神話内の時間**(神代、神武即位前 等)
2. **学術的に推定される歴史時間**(BCE 200〜CE 300 等)
3. **その記述が成立した文献の時間**(古事記=712 等)

これらを一つのカラムに混入させると、後続の解析・可視化・引用が破綻する。本書は **三層を独立カラムで保持する** モデルを規定する。

## 1. 三層の定義

### 1.1 mythic_time(神話時間)

- **意味**: 神話・伝承の **物語内** の時間。
- **書式**: 文字列(列挙語彙)
  - `神代`(国生み・神生み・国譲り 等、皇統開始前)
  - `神武即位前`(東征中など)
  - `神武〜開化朝`(欠史八代)
  - `崇神朝`、`垂仁朝`、`景行朝` …(各天皇朝)
  - `不明`
- **用途**:
  - 神話的事象の物語内位置を記録
  - 「崇神朝の出来事として記載」と「実際にCE 200頃の事件」を別カラムに分離
- **扱い方**:
  - 神話的事象は **mythic_time が必須**
  - 史料的事象は基本 `不明`(神話内に位置づけられない場合)

### 1.2 estimated_historical_time(推定歴史時間)

- **意味**: **学術的に推定される** 西暦範囲。
- **書式**:
  - `CE NNN-NNN`(範囲)
  - `BCE NNN`(紀元前)
  - `不明`
  - 単一年でも `CE 716` のように西暦表記を統一
- **用途**:
  - 神話事象が **実際に起きたとすれば** いつ頃か(対応可能性の幅)
  - 歴史事象の確度ある推定
- **扱い方**:
  - 神話的事象では多くが「不明」または広範囲(`CE 200-600` など)
  - 確度が低い場合は `notes` で議論を補足

### 1.3 document_written_time(文献成立時間)

- **意味**: その記述が **書かれた文献** の成立年。
- **書式**: `CE NNN`(成立年)、または `CE NNN以降`(写本期間がある場合)
- **用途**:
  - その記述自体がいつの時代の認識を反映しているか
  - 文献ごとの記述差(古事記 vs 書紀 vs 風土記 vs 神賀詞)を可視化
- **扱い方**:
  - 同じ事象が複数文献で記述される場合、文献ごとに別レコード or relation で管理
  - 一つの事象に複数文献が言及する場合は、関係性 `mentioned_in` で各文献に紐付け

## 2. 既存スキーマ(編 TSV)との対応

既存の出雲編 TSV では「時代/推定年代/西暦/和暦」が混在していた。これを三層モデルに整理すると:

| 既存カラム | 対応する三層 |
|---|---|
| 時代(神代/旧石器/縄文/弥生/古墳/飛鳥/奈良/平安/鎌倉) | mythic_time(神代の場合)+ estimated_historical_time(時代区分の年代に変換) |
| 推定年代 | estimated_historical_time |
| 西暦 | estimated_historical_time(数値化) or mythic_time が「神代」なら「不明」 |
| 和暦 | document_written_time の補助(成立年の和暦) |

### 移行戦略

既存編 TSV はカラム数を増やさず、解釈で三層を読み取る方針。新規マスター・関係性 TSV では三層を独立カラム化。

## 3. 具体例で整理

### 3.1 国譲り

| 三層 | 値 |
|---|---|
| mythic_time | `神代`(神武即位前の神代後期) |
| estimated_historical_time | `不明`(対応可能性として `CE 200-600` 銅鐸祭祀終焉〜古墳期説 L2 仮説) |
| document_written_time | `CE 712`(古事記)、`CE 720`(日本書紀)、`CE 716以降`(出雲国造神賀詞奏上) |

→ 三層が **完全に異なる時間** を指す。一つのカラムに統合不能。

### 3.2 天孫降臨

| 三層 | 値 |
|---|---|
| mythic_time | `神代`(国譲り直後) |
| estimated_historical_time | `不明`(史実的対応説なし) |
| document_written_time | `CE 712`(古事記)、`CE 720`(日本書紀) |

→ estimated_historical_time が学術的に「不明」と書く重要性。

### 3.3 神武東征

| 三層 | 値 |
|---|---|
| mythic_time | `神武即位前`(東征は神武即位の直前) |
| estimated_historical_time | `不明`(神武即位 BCE 660 は皇紀の基準点だが学術的根拠なし。3世紀末〜4世紀説、6世紀説など諸説 L2-L4) |
| document_written_time | `CE 712`(古事記)、`CE 720`(日本書紀) |

→ 「神武即位 BCE 660」は **document_written_time の世界観** であり、estimated_historical_time ではない。混同が頻発する例。

## 4. 時間軸を扱う関係性

### 4.1 valid_from / valid_until

関係性が時間幅を持つ場合、`relations.tsv` のオプション項目で記録。

例: 出雲大社の中世スサノオ祭神化期
```
DEI-002 → primary_deity_of → SHR-001
  valid_from: CE 1200(中世前期)
  valid_until: CE 1700(近世以前)
  hypothesis_layer: L1
  notes: 中世神道書での記述。近世以降は DEI-001 大国主に復帰
```

### 4.2 temporal_scope

関係性の時間軸が三層のどれに属するかをタグで明示:
- `temporal_scope=mythic`: 神話内の出来事
- `temporal_scope=estimated`: 推定史実
- `temporal_scope=document`: 文献成立時の認識

---

## 5. 注意点

### 5.1 神武紀年問題

『日本書紀』の神武即位 BCE 660 は **天皇紀年の起点** だが、学術的には 3 世紀末〜4 世紀説などの推定がある。

**運用ルール**:
- mythic_time = 「神武即位元年」(物語内では実時間)
- estimated_historical_time = `不明` または `CE 250-400`(諸説併記、L4)
- document_written_time = `CE 720`(書紀)

### 5.2 神社創建年代

社伝が「神武天皇○年創建」と主張する場合:
- mythic_time = `神武朝`(社伝記述)
- estimated_historical_time = `不明`(学術検証不能)
- founding_legend = 自由記述で社伝内容を保存
- founding_year_estimated = `不明` または推定がある場合のみ記入

### 5.3 文献の成立年の幅

古事記=712、日本書紀=720 は確定的だが、出雲国造神賀詞は「霊亀2年(716)以降に複数回奏上」、出雲国風土記は「天平5年(733)成立」。複数文献を扱う際は **document_written_time を必ず固有名で記録**。

### 5.4 中世以降の祭神交代

「中世にスサノオ祭神化された」のような場合、`primary_deity_of` 関係に `valid_from`/`valid_until` を付与し、時系列を保存。
