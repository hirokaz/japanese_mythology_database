# Precinct Zone Model

## Purpose

Define conceptual zones inside shrine precinct topology.

The goal is not architectural precision.
The goal is to represent:

- ritual progression
- spatial transition
- sacred layering
- movement semantics

inside shrine precincts.

---

# Core Concept

A shrine precinct may be interpreted as layered zones.

Example:

```txt
outer_world
 -> threshold
 -> approach
 -> purification
 -> outer_precinct
 -> inner_precinct
 -> sanctuary_center
```

This is NOT universal.
It is a schematic topology model.

---

# Candidate Zones

| zone_id | meaning |
|---|---|
| outer_world | ordinary external world |
| threshold | transition boundary |
| approach | movement corridor |
| purification | cleansing-related area |
| outer_precinct | public precinct |
| inner_precinct | more sacred interior zone |
| sanctuary_center | symbolic sacred center |
| restricted_zone | access-limited zone |
| ritual_route | formal ritual movement corridor |
| sacred_nature | sacred forest / mountain / water area |
