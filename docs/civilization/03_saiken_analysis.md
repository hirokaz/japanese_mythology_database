## S01. 出雲祭祀圏

### 中心構成

| 軸 | 内容 |
|---|---|
| 中心神社 | 出雲大社(SHR-001)、熊野大社(出雲国一宮としての並立、SHR-002 出雲)、神魂神社(SHR-004) |
| 中心神 | 大国主(DEI-001)、スサノオ(DEI-002)、アメノホヒ(DEI-013、出雲国造祖)、コトシロヌシ(DEI-009)、タケミナカタ(DEI-010、出雲→諏訪) |
| 中心豪族 | 出雲国造(CLN-001、千家・北島の二家)、意宇氏(CLN-002)、神門氏(CLN-003) |
| 中心神話 | 国譲り(MOTIF-001)、ヤマタノオロチ退治(MOTIF-026)、出雲国引き(MOTIF-013)、根の堅州国(MOTIF-017) |

### 支配地域・祭祀

- 古代国: 出雲国(意宇郡・神門郡・島根郡・大原郡・楯縫郡・仁多郡・飯石郡 等)
- 関連祭祀: 神在祭(MOTIF-231)、火継神事(MOTIF-235)、青柴垣神事(MOTIF-236)、諸手船神事(MOTIF-237)
- 重要史料: 出雲国風土記(733)、出雲国造神賀詞(『延喜式』『続日本紀』)、出雲国造家文書

### 3 ネットワーク所属

| ネットワーク | 関与度 | 内容 |
|---|---|---|
| 海洋 | ★★ | 海蛇(セグロウミヘビ)漂着祭祀(神在月)、日本海航路 |
| 山岳・修験 | ★ | 古代は薄い、中世以降熊野・修験との習合あり |
| 製鉄・金属 | ★★★ | **奥出雲の製鉄(踏鞴)中心地**。MOTIF-082 金屋子神、MOTIF-087 出雲たたら、CLN 山内・絲原・桜井 |

→ 出雲は **製鉄ネットワークの中核** で、海洋・山岳には部分参加。

### 政治・神話編集

- 出雲国造神賀詞(MOTIF-007、716初奏上):地方視点神話の朝廷取り込み
- 国譲り神話(中央視点)+ 出雲国引き(地方視点):**視点二重構造**
- 千家・北島の二家分立(中世以降):祭祀権の二重化
- 平田篤胤幽冥論(MOTIF-219):大国主の幽冥司宰職を近世神道思想化

### 考古学的根拠

- 加茂岩倉遺跡(銅鐸 39 個一括出土)
- 荒神谷遺跡(銅剣 358 本・銅鉾 16 本・銅鐸 6 個一括出土)
- 西谷墳墓群(四隅突出型墳丘墓)
- 弥生〜古墳期の独立祭祀圏の物的証拠 → 国譲り神話の対応(L2 仮説、HYP-004)

### 隣接圏との関係

- **中央(伊勢圏)**: 国譲り神話で従属関係を構築。神賀詞奏上で祭祀的服属を表現
- **諏訪圏**: タケミナカタの東遷(神話的移動)
- **三輪山圏**: 大物主=大国主同体論争(中世以降の論点)
- **九州圏**: スサノオ→新羅(曽尸茂梨)の渡来神話

### L4-L5 仮説層

- ヤマタノオロチ製鉄説(HYP-002, MOTIF-084, L4): 神話の政治史的読解
- 銅鐸祭祀の終焉と国譲り神話の対応(HYP-004, L2): 学術的に最も検討される仮説
- 出雲振根殺害=出雲服属史実(L2): 6世紀史実反映説

## S02. 伊勢祭祀圏

### 中心構成

| 軸 | 内容 |
|---|---|
| 中心神社 | 内宮(皇大神宮、SHR-002)、外宮(豊受大神宮、SHR-003)、別宮(荒祭・月讀・伊雑・瀧原・倭姫 等 14 社) |
| 中心神 | アマテラス(DEI-003)、豊受大神(DEI-007)、ツクヨミ(DEI-004) |
| 中心豪族 | 荒木田氏(内宮、CLN-004)、度会氏(外宮、CLN-005)、大中臣氏(中央派遣) |
| 中心神話 | 天孫降臨(MOTIF-002)、垂仁朝伊勢創建(MOTIF-014)、天岩戸(MOTIF-057)、三種神器(MOTIF-004) |

### 支配地域・祭祀

- 古代国: 伊勢国(度会・多気・飯野・安濃・鈴鹿・河曲・三重・朝明・員弁 等)、志摩国
- 関連祭祀: 神嘗祭(MOTIF-117)、新嘗祭(MOTIF-116)、大嘗祭(MOTIF-115)、式年遷宮(MOTIF-243)
- 重要史料: 皇大神宮儀式帳(804)、止由気宮儀式帳(804)、延喜式、倭姫命世記(中世)、神道五部書(中世)

### 3 ネットワーク所属

| ネットワーク | 関与度 | 内容 |
|---|---|---|
| 海洋 | ★ | 志摩の海女・海産物御饌、限定的 |
| 山岳・修験 | ★ | 中世の両部神道による山岳習合あり |
| 製鉄・金属 | ★ | 直接の製鉄拠点ではない |

→ 伊勢は **国家祭祀(神話編集)ネットワークの中核**で、3 ネットワークいずれにも軽度参加。

### 政治・神話編集

- 内宮(アマテラス)→ 皇祖神として皇統正統化(MOTIF-011)の核
- 外宮(豊受)→ 食物神、後に「豊受=国常立尊」とする度会神道(中世)
- 元伊勢候補社(籠神社・檜原神社 等)→ 伊勢以前の祭祀地論争(L2)
- 両宮論争(中世):内外二宮の主従論争
- 神道五部書(『天照坐伊勢二所皇太神宮御鎮座次第記』ほか):中世伊勢神道の形成

### 考古学的根拠

- 皇學館近隣の祭祀遺跡(古墳期)
- 元伊勢候補(瀧原宮遷御伝承、磯部宮など)

### 隣接圏との関係

- **出雲圏**: 国譲りで上位、神嘗祭で皇祖神祭祀を独占
- **三輪山圏**: 倭姫巡幸の出発地は三輪 → 国家祭祀の地理的二項
- **熊野圏**: 中世伊勢神道で本地垂迹的習合

### L4-L5 仮説層

- 元伊勢諸候補の歴史的経路(L2)
- 両宮論争の歴史的根拠(L2)
- 「アマテラス男神説」(中世密教神話、L4)

## S03. 諏訪祭祀圏

### 中心構成

| 軸 | 内容 |
|---|---|
| 中心神社 | 諏訪上社本宮(SHR-039)・上社前宮(SHR-040)・下社秋宮(SHR-041)・下社春宮(SHR-042)、八剣神社(SHR-043) |
| 中心神 | タケミナカタ(DEI-010、出雲→諏訪)、八坂刀売(DEI-011)、洩矢神(DEI-012、在来神)、ミシャグジ(DEI-013'、土着信仰) |
| 中心豪族 | 諏訪氏(大祝、CLN-017)、守矢氏(神長官、CLN-018)、金刺氏(下社、CLN-019) |
| 中心神話 | 諏訪洩矢神戦争(MOTIF-201)、諏訪龍蛇神(MOTIF-030)、諏訪御渡(MOTIF-036)、ミシャグジ(MOTIF-146)、洩矢神(MOTIF-147) |

### 支配地域・祭祀

- 古代国: 信濃国諏訪郡(諏訪盆地・八ヶ岳南麓)
- 関連祭祀: 御柱祭(MOTIF-232、7年式年)、御頭祭(MOTIF-233/120、鹿頭75)、蛙狩神事(MOTIF-234、元日)、薙鎌神事
- 重要史料: 守矢家文書、諏訪上社縁起、諏方大明神画詞(中世)

### 3 ネットワーク所属

| ネットワーク | 関与度 | 内容 |
|---|---|---|
| 海洋 | ★ | 直接的接続なし(内陸盆地) |
| 山岳・修験 | ★★★ | **山岳信仰の中核**。諏訪+八ヶ岳+守屋山+御射山 |
| 製鉄・金属 | ★ | 諏訪洩矢神戦争での「藤刀 vs 鉄輪」象徴(製鉄民征服?L3) |

→ 諏訪は **山岳ネットワークの中核**で、縄文系信仰(ミシャグジ)を保存する独立性が高い。

### 政治・神話編集

- タケミナカタ:国譲りで敗れて諏訪に逃避(古事記)→ 神話的東遷
- 諏訪在来神(洩矢神)を **古事記が無視**、地方伝承(守矢家文書)で保存
- 守矢氏(神長官):縄文期からの祭祀継承を主張する地方氏族
- 大祝(諏訪氏)と神長官(守矢氏):**祭祀権の二重化**(中世)
- ミシャグジ:全国に「御社宮司」「シャグチ」として分布、諏訪を中心とする縄文系祭祀の生証拠説(L3)

### 考古学的根拠

- 八ヶ岳南麓の縄文中期遺跡群(井戸尻・尖石・棚畑など)
- 諏訪上社前宮周辺の縄文祭祀遺構
- 御柱と縄文期柱建ての連続性(L3 仮説)

### 隣接圏との関係

- **出雲圏**: タケミナカタの東遷神話(国譲り)
- **東北圏**: ミシャグジ↔アラハバキの縄文系信仰仮説連関(L3)
- **熊野圏**: 諏訪修験の参入

### L4-L5 仮説層

- 御柱の縄文起源(L3)
- ミシャグジ=縄文期信仰の生存(L3)
- 洩矢神=縄文系先住神(L3)
- 「藤刀 vs 鉄輪」=金属技術征服譚(L3-L4)
