# 解析方針 — 祭祀OSとしての日本列島

## 1. 「祭祀OS」という視点

日本列島の古代〜中世の社会構造は、**祭祀ネットワーク** によって編成されていた。

- **神を祀る権利** = **土地支配権**(国造の神社祭祀、社家の世襲権)
- **神話編集権** = **王権の正統化**(古事記書紀、神賀詞、各社縁起)
- **神社ネットワーク** = **物流・情報・婚姻ネットワーク** とほぼ重なる
- **山岳信仰** = **流域支配・水利支配** の宗教化
- **海洋信仰** = **航路支配・国際交易** の宗教化
- **製鉄ネットワーク** = **鉄資源支配** の宗教化(鍛冶神・金山彦)

これら全体を「OS(オペレーティングシステム)」と呼ぶのは、それらが日本列島の **諸活動を駆動する基盤層** として機能していたから。

## 2. 解析の三層

| 層 | 名称 | 内容 |
|---|---|---|
| 上層 | 神話 | モチーフ・物語の網、誰がどう編集するか |
| 中層 | 制度 | 神社・社格・氏族・祭祀職、誰が祭る権利を持つか |
| 下層 | 物質 | 神体山・磐座・古墳・遺跡、物質的な祭祀基盤 |

各層は独立に分析するが、**相互参照** により全体構造が見える。

| 神を祀る権利 | 土地支配 |
|---|---|
| 出雲国造の出雲祭祀権 | 出雲国の祭政一致 |
| 三輪氏の大物主祭祀権 | 三輪山周辺の地域支配 |
| 阿蘇氏の阿蘇神社祭祀権 | 阿蘇カルデラの支配 |
| 宗像氏の沖ノ島祭祀権 | 玄界灘海路の支配 |
| 諏訪氏の諏訪上社祭祀権 | 諏訪地方の支配 |

→ 神社=領主の祭祀拠点、神=領主の正統性根拠

## 5. 重要原則

- 神話を単純否定しない
- 仮説を断定しない
- 学術説と地方伝承を区別
- 地方祭祀を重視
- 神仏習合を重視
- relation 構造を重視
- 「神を祀る権利」が政治権力と結びついていた視点を重視
- 「祭祀OS」として日本列島を解析する視点を重視
