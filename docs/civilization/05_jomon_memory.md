# 縄文記憶仮説体系

本書は **縄文期(BCE 14000〜BCE 300 頃)の祭祀・文化的記憶が日本神話・民俗信仰にどう残存するか** を、L0〜L5 の仮説層で厳格に区別して整理する。

> 関連: `docs/civilization/00_methodology.md`(祭祀OS論)、`docs/civilization/03_saiken_analysis.md` §10.4(縄文記憶残存度)、`docs/civilization/04_networks.md`(山岳系の縄文連続)

> **重要原則**: 縄文記憶仮説は **検証困難な領域**(L3-L5)を多く含む。本書は仮説層の混同を避けるため、各論ごとに layer を厳格に付与する。

## 1. 縄文記憶仮説とは

### 1.1 定義

**縄文記憶仮説**:縄文期の祭祀・自然観・地形認識が、後世(弥生・古墳・歴史時代)の神話・民俗信仰に **痕跡として保存** されているとする一群の仮説。

### 1.2 主要争点

| 軸 | 内容 |
|---|---|
| **時代差** | 縄文(14000年間) vs 後世神話文献(8世紀〜)の間の連続性をどう論証するか |
| **断絶論** | 弥生稲作文化の渡来で縄文文化は **断絶** したとする立場(L1) |
| **連続論** | 縄文層が地方祭祀・民俗の **基層** として残存したとする立場(L3-L4) |
| **検証困難性** | 文字記録がない縄文期の祭祀内容は **物的考古学のみで推測** するしかない |

### 1.3 仮説層の厳格化

| layer | 取扱方針 |
|---|---|
| L0 | 縄文期の **物的事実** (遺跡・遺物の存在自体) |
| L1 | 考古学的に **広く認められる** 推定 (土偶=祭祀的意味) |
| L2 | 複数研究者が言及する **推定** (環状列石=葬祭儀礼) |
| L3 | **地方伝承との対応**(ミシャグジ=縄文系祭祀残存等) |
| L4 | **大胆な仮説**(縄文海進記憶、アラハバキ縄文起源等) |
| L5 | **検証困難だが思想的に興味深い**(縄文ユートピア論等) |

## 2. 海面上昇仮説(縄文海進)

### 2.1 物的事実(L0)

- **縄文海進**:BCE 6500-5000 頃、海面が現在より +2〜+5m 上昇
- 関東平野は海(古東京湾)、瀬戸内海は内海化、北海道〜関東に縄文遺跡が標高高めに分布
- 出雲平野(古宍道湖は広大)、関西平野も内海化

### 2.2 神話との対応(L4-L5)

| 神話 | 海進記憶仮説 | layer |
|---|---|---|
| 出雲国引き(MOTIF-013) | 縄文期の海面記憶が国引き神話に転化(海岸線変動の人格化) | L4 |
| 国生み(MOTIF-003) | 列島誕生神話=縄文海進前の地形からの島生成 | L5 |
| 黄泉国(MOTIF-016) | 海没した縄文集落の冥界化 | L5 |
| ダイダラボッチ(MOTIF-152) | 巨人の地形改変=海進期の地形変動の人格化 | L5 |

→ いずれも **検証困難**。HYP-001(縄文海進記憶仮説)として hypothesis_master に登録予定。

### 2.3 考古学的根拠

- 古東京湾の縄文貝塚分布(海岸線の指標)
- 古宍道湖の海退過程(出雲平野形成史)
- 三内丸山遺跡(青森)の繁栄期は海進期

### 2.4 反証・批判

- 神話記述は8世紀文献。**8000年以上の口承伝達** は不可能とする批判(L1 立場)
- 縄文海進の地形変動は **緩慢**(数千年スケール)で、特定神話化されにくい(L1)
- 国引き神話は弥生〜古墳期の **新羅・北陸・隠岐との海路認識** で十分説明可能(L1-L2)

### 2.5 現代の評価

- 学術的には **「興味深いが論証不可能」**(L4-L5)
- 地方伝承(諏訪・東北)との総合により思想史的価値は認められる
- 折口信夫「まれびと」論、柳田國男民俗学の延長として位置づける研究者もいる(L3-L4)

## 3. 蛇神原型・土偶女神論・縄文祭祀残存

### 3.1 蛇神原型

#### 3.1.1 物的事実(L0)

- 縄文中期(BCE 3000-2000)、蛇文土器(火炎土器の蛇形装飾、新潟・長野等)
- 縄文中期、蛇形土製品(諏訪地方、井戸尻遺跡群)
- 弥生期、銅鐸文様の蛇形(複数事例)

#### 3.1.2 連続性仮説(L3-L4)

| 仮説 | 内容 | layer |
|---|---|---|
| 縄文蛇神→弥生銅鐸蛇文 | 蛇神祭祀の物質的継承 | L3 |
| 縄文蛇神→大物主(三輪山) | MOTIF-035 大物主の蛇身、L3-L4 |
| 縄文蛇神→ミシャグジ | MOTIF-146、MR-SY-058、L3 |
| 縄文蛇神→水神=蛇 | MOTIF-034、L3-L4 |
| 縄文蛇神→白蛇信仰 | MOTIF-037、L3-L4(中世以降に継承) |

#### 3.1.3 反証

- 蛇形装飾は **狩猟採集民に普遍的**(東アジア・北米先住民等)。日本独自連続性とは限らない(L1)
- 銅鐸は弥生期(渡来文化)。縄文の連続というより **新規導入** が主(L1)

### 3.2 土偶女神論

#### 3.2.1 物的事実(L0)

- 縄文期、推定18,000体以上の土偶出土
- 多くは **女性形**、乳房・妊娠表現
- 中期後半〜後期に大型化(縄文のヴィーナス、合掌土偶、遮光器土偶 等)
- 故意の **破壊** が多く見られる(再生・治癒祭祀の可能性 L1-L2)

#### 3.2.2 女神論仮説(L2-L4)

| 仮説 | 内容 | layer |
|---|---|---|
| 土偶=女神信仰 | 大地母神への信仰の物的証拠 | L2 |
| 土偶=破壊再生祭祀 | 病気平癒・豊穣祈願 | L1-L2 |
| 土偶→縄文女神→アマテラス系 | 太陽女神への系譜 | L4 |
| 土偶→縄文女神→ミシャグジ女神性 | 諏訪在来神の女神論 | L4 |
| 土偶→イザナミ(国生み女神) | 大地母神の連続 | L4-L5 |

#### 3.2.3 反証

- 土偶=女神 は **断定不能**。男性形・抽象形・動物形も存在(L1)
- 縄文の宗教観は文字資料なし、女神論は **近代解釈** の投影(L1)

### 3.3 縄文祭祀の残存

#### 3.3.1 物的事実(L0)

- 環状列石(大湯・忍路 等):縄文後期、葬祭儀礼の場(L1)
- 環状木柱列(チカモリ・真脇 等):縄文晩期、巨大柱建ての儀礼遺構(L1-L2)
- 配石遺構・立石(ストーンサークル系):全国
- 黒曜石・翡翠の交易ネットワーク(縄文中期):広域祭祀ネットワーク

#### 3.3.2 後世への連続性仮説

| 縄文要素 | 後世への連続(仮説) | layer |
|---|---|---|
| 環状木柱列(チカモリ・真脇) | 諏訪御柱(MOTIF-136) | L3 |
| 環状列石(大湯) | 神奈備山周辺の祭祀(MOTIF-068) | L3 |
| 配石・立石 | 磐座(MOTIF-131) | L3 |
| 黒曜石交易 | 古代海人ネットワークの原型 | L2 |
| 山岳祭祀 | 神体山信仰(MOTIF-067) | L3 |
| 火祭祀(火炎土器) | カグツチ系火神 | L4 |

#### 3.3.3 諏訪における強い継続性(L3)

- 八ヶ岳南麓の縄文中期遺跡集中(井戸尻・尖石・棚畑 等)
- 諏訪上社前宮周辺の縄文祭祀遺構
- ミシャグジ・洩矢神・御柱の縄文起源仮説(MR-EV-087, MR-EV-088)
- 守矢家文書による祭祀継承の主張(中世以降の家伝、L2 史料的妥当性)

→ 諏訪は **縄文記憶仮説の最有力候補地**(本書 §3 で詳述)
