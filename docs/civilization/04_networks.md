# 海上・山岳・製鉄ネットワーク統合解析

本書は **海上交通 / 山岳修験 / 製鉄鍛冶** の 3 ネットワークを統合的に解析する。各ネットワークは『祭祀OS』(`docs/civilization/00_methodology.md`) の中層〜下層に対応する。

> 関連: `docs/civilization/03_saiken_analysis.md`(9 地域祭祀圏)、`docs/civilization/04_networks.tsv`(集計)

### 1.4 考古学的根拠

- 沖ノ島祭祀遺跡(4-9世紀、8万点遺物、世界遺産2017)

- 安曇族の北部九州→信濃移動(地名分布)

### 1.5 政治的読解

- **国家祭祀の対外保証**:沖ノ島祭祀=ヤマト王権の対朝鮮交渉ロジ(L1-L2)
- **海人族の系譜化**:海神を祖とする氏族化(L0)
- **アマテラス・スサノオ誓約段**:海神族と皇統の血縁的接続(L0)

### 1.6 縄文・弥生段階(L3-L5)

- 縄文海進記憶(L5、HYP-001)
- 海食洞祭祀(縄文後期〜弥生)
- 黒曜石・翡翠の海上交易ネットワーク(縄文期、L1-L2 考古学)

### 1.7 山岳・製鉄ネットワークとの重なり

- **熊野**:海洋(★★) + 山岳(★★★)= 補陀落渡海+熊野修験
- **九州**:海洋(★★ 隼人) + 山岳(★★★ 阿蘇/霧島/英彦山) = 火山+海洋系
- **出雲**:海洋(★★ 海蛇) + 製鉄(★★★ たたら)= 神在月+鍛冶神

- **役小角**(役行者、7世紀):修験道の祖、葛城→吉野→熊野→大峰
- **平安期**:本地垂迹の体系化(MOTIF-166)

- **近世**:富士講・御嶽講などの民間宗教化
- **明治神仏分離令**:修験道廃止令(1872)で大打撃 → 戦後一部復興
