# 中央史観 vs 地方祭祀構造

本書は **中央(ヤマト王権・朝廷)による神話正統化の機制** と **地方独自祭祀の残存・抵抗・包摂** を、対の構造として整理する。

> 関連: `docs/civilization/05_jomon_memory.md`(縄文記憶仮説)、`docs/civilization/03_saiken_analysis.md` §10.3(中央⇄地方の二項構造)、`docs/audit/08_regional_audit.md`(地方軽視監査)

> **重要原則**: 本書は「中央=悪、地方=善」のような価値判断ではなく、**両者の制度的・神話編集的関係** を構造分析する。地方独立性を尊重しつつ、中央祭祀の歴史的役割も認識する。

## 1. 中央正統化の機制

### 1.1 神話編集権による正統化

| 機制 | 具体事例 | 史料 |
|---|---|---|
| **国家神話編纂** | 古事記(712)、日本書紀(720) | 各書 |
| **地方神話の中央化** | 出雲国引き(風土記独自) → 国譲り(記紀標準) | 出雲国風土記 vs 古事記 |
| **氏族系譜の整理** | 新撰姓氏録(815)、先代旧事本紀(平安初期) | 各書 |
| **神祇制度の確立** | 延喜式(927)、神名帳の式内社制度 | 延喜式 |
| **皇祖神の独占祭祀** | 皇大神宮儀式帳(804)、伊勢神宮の制度化 | 皇大神宮儀式帳 |
| **神賀詞の朝廷儀礼化** | 出雲国造神賀詞(716初奏上) | 続日本紀、延喜式 |

### 1.2 物的祭祀権の集約

| 機制 | 具体事例 |
|---|---|
| **三種神器の独占** | 八咫鏡(伊勢)、草薙剣(熱田)、八尺瓊勾玉(皇都) |
| **国家祭祀施設の整備** | 伊勢神宮(垂仁朝伝)、各国一宮制度 |
| **神宝の中央集権化** | 物部氏の七支刀(石上神宮、CE369) |
| **奉幣制度** | 延喜式神名帳の式内社、22 社奉幣 |

### 1.3 神話成立年代と神話の描く時代の二重性

- **神話成立年代**:8 世紀前半(古事記712 / 書紀720)
- **神話の描く時代**:神代〜推古朝(8 世紀以前数百年〜数千年)
- → **8 世紀の編纂意図** が神話に反映:皇統正統化、対外関係(対朝鮮)、対地方(出雲・諏訪・宗像 等)

→ CLAUDE.md §4 の「神話成立年代と神話描写時代の混同禁止」は、この二重性を踏まえた原則。

### 1.4 中央正統化のパターン分類

#### A. 地方神を皇統に組み込む(系譜化)

| 地方神 | 系譜化結果 | 史料 |
|---|---|---|
| 大国主(出雲) | スサノオの系譜下、皇統に幽冥司宰として位置づけ | 古事記 |
| 大物主(三輪山) | 大国主和魂、崇神朝に祀られる | 書紀崇神紀 |
| アメノホヒ(出雲) | 出雲国造の祖、アマテラスの子 | 古事記 |
| ニニギ系神統(九州) | 皇祖神の地上展開 | 古事記 |

#### B. 地方神を悪役・敗者として描く(神話化された服属)

| 地方神 | 神話的扱い | 史料 |
|---|---|---|
| ヤマタノオロチ(出雲) | スサノオに退治される悪神 | 古事記 |
| タケミナカタ(出雲→諏訪) | 国譲りで敗れて諏訪へ逃避 | 古事記 |
| 土蜘蛛(全国) | 蔑称化された先住民 | 諸風土記 |
| 蝦夷(東北) | 征討対象 | 続日本紀 |
| 隼人/熊襲(南九州) | 服属対象、海幸末裔として整理 | 書紀 |

#### C. 地方神を独立に保つ(吸収しきれない地域)

| 地方神 | 残存形態 | 地域 |
|---|---|---|
| ミシャグジ | 独立神として全国分布 | 諏訪・全国 |
| 洩矢神 | 守矢氏祖、諏訪在来層 | 諏訪 |
| アラハバキ | 客人神化、東北で独立 | 東北・関東 |
| 健磐龍命 | 阿蘇火山神、ニニギ系に半併合 | 肥後 |

#### D. 地方神を中央祭祀へ組込み(神賀詞型)

| 地方神 | 組み込み形 | 史料 |
|---|---|---|
| 大国主+出雲国造 | 神賀詞奏上(出雲祭祀の朝廷化) | 続日本紀、延喜式 |
| 物部氏祖神 | 石上神宮(国家祭祀の主要社) | 延喜式 |
| 中臣氏祖神 | 春日大社(藤原氏氏神→国家祭祀化) | 延喜式 |

### 1.5 中央正統化の限界

- **諏訪のミシャグジ・洩矢神**:古事記が無視、地方文献(守矢家文書)で保存
- **東北のアラハバキ**:中央史料に登場しない
- **南九州の在地祭祀**:「隼人」「熊襲」と他称化のみで内部は記述されない
- **琉球・蝦夷**:古代史料圏外

→ **中央史観で語られない地方層** が、地方伝承・社伝・民俗で残存(CLAUDE.md §4「地方軽視を回避」)

## 2. 地方残存のパターン

### 2.1 文献保存型

- 出雲国造神賀詞:朝廷儀礼化された **地方視点神話** の保存
- 各国風土記:733 年和銅 6 年詔による地方伝承の文字化命令
  - 出雲国風土記(完本残存):中央神話と異なる独自エピソード(国引き等)
  - 播磨・常陸・豊後・肥前(残欠):独自神話の断片
- 籠神社海部氏系図:海部氏の家伝(国宝、丹後)

### 2.2 社伝・縁起型

- 諏訪大明神画詞(中世):諏訪在来層の保存
- 守矢家文書:神長官家の中世以降祭祀継承記録
- 阿蘇神社縁起:健磐龍命神話の保存
- 各社縁起:中世以降の地方独自神話の集積

### 2.3 民俗・地方伝承型

- 遠野物語(柳田國男 1910):東北民俗の集積
- 各地方の口承伝承:落人伝承、山伏伝承、海人伝承
- 折口信夫『古代研究』:民俗学的読解

### 2.4 偽書型(注意)

- 東日流外三郡誌:偽書とされる(L5、ほぼ偽書認定)
- 竹内文書:偽書(超古代史)
- → **信頼層から分離**(`docs/audit/07_assertion_audit.md` 参照)

## 3. 中央⇄地方の二項構造の代表事例

### 3.1 出雲

| 中央視点 | 地方視点 |
|---|---|
| 国譲り(従属神話、古事記) | 国引き(独立創成神話、出雲国風土記) |
| ヤマタノオロチ退治(スサノオの英雄譚) | 出雲祭祀の物質的中心(銅鐸文化圏) |
| 出雲国造=アメノホヒ系(皇統下) | 国造神賀詞=独自視点の保存 |

### 3.2 諏訪

| 中央視点 | 地方視点 |
|---|---|
| タケミナカタ=出雲系敗者 | タケミナカタ=諏訪入り、八坂刀売との婚姻 |
| (洩矢神に言及なし) | 洩矢神=在来層、諏訪洩矢神戦争 |
| (ミシャグジに言及なし) | ミシャグジ=守矢氏祭祀、御頭祭 |

### 3.3 東北

| 中央視点 | 地方視点 |
|---|---|
| 蝦夷=征討対象、土蜘蛛=蔑称 | 阿弖流為=英雄、独自首長層 |
| (アラハバキに言及なし) | 東北客人神、縄文系神格説 |

### 3.4 南九州

| 中央視点 | 地方視点 |
|---|---|
| 隼人=服属対象、海幸末裔 | 隼人独自系譜、霧島・阿蘇祭祀 |
| 熊襲=ヤマトタケル征討対象 | 在地豪族としての独自性 |

## 4. 結論(S05 段階)

1. **中央正統化は 4 パターン**(系譜化・敗者神話化・独立保持・中央祭祀組込)
2. **地方残存は 4 媒体**(文献・社伝・民俗・偽書)
3. **二項構造が顕著なのは出雲・諏訪・東北・南九州**
4. **中央史観で語られない層** を尊重するため、本プロジェクトは relation で **両視点を併記**(MR-PO-007 等)
5. **偽書を信頼層から分離**(L5)することで、地方伝承の史料価値を保護
