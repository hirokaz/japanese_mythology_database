## 1. 神話編集ネットワーク(新規独立)

### 1.1 構造

朝廷・国造・社家・中世神道書圏が **神話の編集権・正統化権** を保持する非物的ネットワーク。

### 1.2 中軸拠点

- 伊勢神宮(皇祖神祭祀の独占)
- 大和朝廷(古事記・書紀編纂)
- 出雲国造(神賀詞奏上の独自視点)
- 物部氏(石上神宮+先代旧事本紀)
- 中臣・忌部氏(古語拾遺)
- 中世神道書圏(伊勢神道・吉田神道・両部神道・山王神道)
- 近世国学(本居宣長→平田篤胤)
- 近代国家神道(明治政府)

### 1.3 編集権の連鎖

```
古事記(712, 太安万侶) → 書紀(720, 舎人親王)
   ↓
風土記(733〜, 各国)
   ↓
新撰姓氏録(815) + 先代旧事本紀(平安初期)
   ↓
延喜式(927)
   ↓
中世神道書(13-15C)
   ↓
古事記伝(1798, 本居宣長)
   ↓
平田篤胤幽冥論
   ↓
近代国家神道(明治〜昭和)
   ↓
1872 神仏分離令
```

### 1.4 motif politicalization の典型

- 国譲り(中央版)→ 国引き(地方版)の対立
- 大物主祟り → 崇神朝祭祀整備による政治的固定
- 神功皇后三韓征討 → 八幡神顕現 → 全国八幡信仰
- 天孫降臨 → 皇統正統化(8 世紀編纂)

## 2. 神仏習合ネットワーク(新規独立)

### 2.1 構造

本地垂迹・修験権現・密教習合の **編集権** を保持する宗教思想ネットワーク。

### 2.2 中軸拠点

- 熊野三山(熊野権現+本地仏)
- 宇佐 → 石清水 → 鶴岡(八幡大菩薩)
- 春日大社+興福寺(春日権現)
- 比叡山日吉(山王神道)
- 高野山(両部神道)
- 出羽三山(月山権現+羽黒権現+湯殿権現)
- 大峰・吉野(蔵王権現)
- 白山(白山妙理権現)

### 2.3 習合の典型パターン

- スサノオ ↔ 牛頭天王(疫神)
- 大国主 ↔ 大黒天
- 市杵島姫 ↔ 弁才天
- 八幡神 ↔ 阿弥陀如来 ↔ 応神天皇
- 熊野権現:家都美御子 ↔ 阿弥陀、速玉 ↔ 薬師、夫須美 ↔ 千手観音
- 蔵王権現:釈迦・観音・弥勒の三尊
- 白山妙理 ↔ 十一面観音
- 富士浅間(コノハナサクヤ) ↔ 大日如来(中世)

### 2.4 1872 神仏分離令による断絶

- 蔵王権現 → 金峯神社へ強制改称
- 牛頭天王 → スサノオへ祭神変更
- 春日権現 → 春日大神(神道専一)
- 八幡大菩薩 → 八幡大神/応神天皇
- 多くの修験霊場が解体・神社化

## 6. ヤマト王権の 5 方向掌握モデル

```
                     [神話編集] (伊勢中軸)
                          │
                          ↓
                   ┌─ 古事記・書紀編纂(8世紀)
                   │  皇統正統化
                   │  地方神話の中央化
                   │
[海上] ─→ 沖ノ島祭祀 → 対朝鮮交渉ロジ
   │       (4-9世紀)
   │
[山岳] ─→ 熊野・大峰・出羽 → 修験道 → 中世以降に独立化
   │       (中央接続から地方独自展開へ)
   │
[製鉄] ─→ 出雲・吉備 → 神話化(オロチ・温羅) + 物的支配
   │
[神仏習合] ─→ 熊野・八幡・春日 → 中世神道書 → 国家神道(逆方向掌握)
                          ↑
                  [1872 神仏分離令で強制終了]
```

→ ヤマト王権は **5 方向掌握** で対外・宗教・軍事・神話編集・思想統制を一手化。1872 で神仏習合のみ強制終了、他軸は近代まで継続。
