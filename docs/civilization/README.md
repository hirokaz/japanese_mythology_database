# 祭祀文明構造解析

本ディレクトリは「日本列島における祭祀ネットワークと権力構造の変遷」を **構造化** することを目的とする。

## 視点

「祭祀OS としての日本列島」という分析視点を採用する。

- 神話 / 神社 / 豪族 / 皇統 / 祭祀 / 地域支配 / 山岳 / 海洋 / 製鉄 / 修験 / 神仏習合
- これらが **巨大ネットワーク** として結びついていた構造を解析する
- 「神を祀る権利」が政治権力と結びついていた視点を最重視
